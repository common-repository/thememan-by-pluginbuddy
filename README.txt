ThemeMan by PluginBuddy.com helps you manage a large number of themes by allowing quick previewing and activation of themes from the theme manager within the dashboard.

== Description ==

ThemeMan, short for Theme Manager, adds a drop-down list to the Themes page in the WordPress administrator Dashboard allowing you to rapidly preview or activate different themes on your site. This is particularly useful for power users and developers looking to preview or work with a multitude of themes.

== Screenshots ==

1. A list of all available themes if added to a drop-down menu at the top of your `Manage Themes` page within the dashboard.  You can select any theme to Preview or Activate quickly.

== Installation ==

The automatic plugin installer should work for most people. Manual installation is easy and takes fewer than five minutes.

1. Upload the ThemeMan directory from the ThemeMan ZIP file into your plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Follow any other on-screen directions.
